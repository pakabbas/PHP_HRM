# UI / Frontend Design Guidelines - Professional Blue & White Theme

Overall Style:
- Clean, corporate, modern, bold visual identity.
- Blue and white as primary theme.
- High-contrast typography, organized layout, calm interface.
- Minimal clutter, strong alignment, generous spacing.

Color Palette:
- Primary Blue: #0A84FF
- Dark Blue: #003B73
- Light Blue: #E6F2FF
- White: #FFFFFF
- Gray Text: #6B7280
- Dark Text: #1A1A1A
- Danger Red: #FF3B30
- Success Green: #27AE60

Layout Structure:
- Left sidebar navigation (solid white background with icons + text)
- Top navigation bar (blue with white text)
- Content area: white cards on light-blue background
- Dashboard cards: bold numeric metrics with icons
- Tables: striped rows, borders subtle, hover highlight row
- Buttons: rounded 8–12px, flat color, bold text

Typography:
- Font: Inter / Roboto / SF Pro
- Headings: bold and large, clear hierarchy
- Body text: 14–15px readable
- Table headings: uppercase small tracking

Buttons:
- Primary (action): background #0A84FF, text white
- Secondary: outline blue, text blue
- Danger: red background
- Hover: slightly darker shade + soft elevation

Required Screens:
- Login page (minimal, centered, modern)
- Dashboard (metrics cards, recent leave requests, attendance summary)
- Employees (list/add/edit/details)
- Attendance (list/import excel)
- Leave requests list + apply + approve/reject
- Payroll center (salary settings, allowances, deductions, run payroll)
- Reports with export buttons
- Configurations & Users

UI Components:
- Dashboard metric cards with icon top-right
- Floating quick-action button (blue circle with +)
- Notification toast system
- Confirmation modal for delete/approval
- Search bar and filters above tables
- Pagination bottom-right

UX Rules:
- Clear and fast navigation
- Minimal text, more structured design
- Consistent spacing (24px sections, 16px inner padding)
- Icons used for actions instead of text
- Breadcrumbs for navigation awareness

Responsive Rules:
- Mobile sidebar collapses to hamburger top-left
- Cards stack vertically on mobile
- Tables become scrollable horizontally
- Full-width forms on small screens

Brand Tone:
Modern · Professional · Corporate · Appealing · Clean · High Usability
